##############################################
使い方
##############################################

プロジェクトフォルダ直下にある init.sample.php というファイルを
ご自身の環境にあわせて書き換え、init.php と名称変更してアップロードしてください。

init.php にもありますが、このブログを機能させるためには
fp_common_modules と、コメント機能を使用するにはさらに PHP HP BBS が必要です。
https://enin-world.sakura.ne.jp/ のアプリページよりダウンロードして解凍し、
お使いのレンタルサーバーにアップロードしてください。

また、init.php と同じように
admin_init.sample.php を admin_init.php に名称変更して
アップロードしてください。

このファイルに管理ページ認証用のユーザー名とパスワードを記載します。
パーミッション設定は [600] などにしてください。



▼ 記事の投稿について

管理用ページから投稿することもできますが、
[プロジェクトフォルダ]/articles/ というディレクトリに txt ファイルを作って
FTP でアップロードすることもできます。
ファイル名は「西暦4桁 + 月2桁 + 日2桁」の、計8桁の半角数字にしてください（例：20220101.txt）。
違う名前にするとバグります。

その場合、lists フォルダ の articles.txt にデータを追加してください。
1行につき、1記事のデータです。
書式についてですが、カテゴリ1|カテゴリ2|8桁の日付数字|タイトル|日時|0 という感じに半角の縦棒で区切ってください。
（例： 2|3|20211231|去年のタイトルです|2021-12-31_09:33:33|0 ）
なお最後の 0 は改行タグの混入を防ぐためのただのおまけです。



▼ カテゴリについて

lists フォルダの categories.txt というファイルを編集します。
書式についてですが、カテゴリ名|カテゴリ名（英数字のみ）|0 という感じです。
最後の 0 は articles.txt と同様バグ防止用のおまけです。



▼ HTML タグと特殊タグについて

文中で HTML タグを書けば、そのまま有効な HTML タグとして表示されます。

特殊な数字タグを使用することで、文字の大きさを変えられます。
「<5>テスト</5>」といったふうに、文字列を<数字>タグで囲うことで部分的にサイズを調整します。
数字は 1 から 9 まであり、5 が標準、9 が一番大きいです。
各大きさは style.css をいじれば変えられます。

また、<img1 />、<img3 /> などというふうに img の後ろに数字を追加した特殊タグは
img/[日付フォルダ（20220411 などといった 8 桁の数字）] というフォルダに入っている
画像ファイルを、ファイル名順に表示します。

たとえば img_1000.jpg と img_1001.jpg というファイルがフォルダの中に入っている場合、
<img1 /> で前者、<img2 /> で後者を呼び出すことができます。
